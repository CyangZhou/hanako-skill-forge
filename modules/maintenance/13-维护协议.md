# 13-维护协议.md

## 维护目标

- 主来源唯一；
- 引用不断链；
- 术语不漂移；
- 模块职责不重叠；
- 脚本和文档口径一致。

## 维护动作

### 1. 引用巡检
```bash
python scripts/main.py --action audit_refs
```

### 2. 术语巡检
```bash
python scripts/main.py --action audit_terms
```

默认排除 `scripts/main.py` 自身，避免词表在巡检器里自命中。

### 3. 整体健康检查
```bash
python scripts/main.py --action audit_skill
```

## 维护规则

- 每次改模块前，先知道主来源在哪；
- 改入口时，连带检查路由与模板；
- 改脚本时，回看对应文档是否需要同步；
- 发现重复页时，必须收口，不并存；
- 维护动作要留下清单或摘要。
