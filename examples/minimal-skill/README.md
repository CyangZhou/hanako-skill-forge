# Minimal Skill Example

This example shows the smallest useful shape of a reusable skill asset.

## Purpose

Use this as a reference when you want to turn a loose prompt or workflow note into a skill with:

- a clear purpose
- explicit scope
- a readable entry file
- minimal structure

## Suggested minimum

```text
example-skill/
  SKILL.md
```

## What to define first

- what the skill is for
- when it should be used
- what it should not try to do
- what the user should get after using it

## Why it matters

A minimal skill is often better than a large but unclear one.

The goal is not to add more files than necessary, but to create a skill that another person can actually understand and maintain.
